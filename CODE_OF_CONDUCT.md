# Code of Conduct

`dataproduct-kit` is intended to be a practical, respectful open-source project.

Participants are expected to:

- Be direct, constructive, and professional.
- Focus criticism on ideas, code, and behavior rather than people.
- Respect different levels of experience and different operating contexts.
- Avoid harassment, personal attacks, discriminatory language, and unwanted
  attention.
- Keep discussions relevant to the project’s goals.

Maintainers may moderate issues, pull requests, discussions, or contributions
that do not meet these expectations.

If you need to report a conduct issue, contact the maintainers privately through
the repository owner’s GitHub profile.
